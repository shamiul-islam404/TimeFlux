# ⏱ TimeFlux — Time Difference Calculator

> *Measure the distance between moments.*

TimeFlux is a beautifully designed, single-page web app that lets you explore time in every dimension — from precise date differences to live countdowns, timezone conversions, age calculations, and real astronomical sky events. Built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies, no tracking.

---

## ✨ Features

### Δ Diff — Time Difference
Calculate the exact gap between any two moments in history or the future. Get the result broken down into years, months, weeks, days, hours, minutes, and seconds — with a human-readable summary.

### ± Calc — Date Calculator
Add or subtract any amount of time from a date. Useful for deadlines, countdowns, or figuring out what date falls 90 days from now.

### ⊕ TZ — Timezone Converter
Convert any moment across all major world timezones instantly. Includes a live world clock showing the current time in cities around the globe.

### ⏱ Live — Live Countdown
Set a target date and watch a real-time countdown tick down to it — or count upward from a past event. Updates every second.

### 🎂 Age Calculator
Enter your birthdate and get your precise age in years, months, days, and hours. Includes a live countdown to your next birthday.

### 🌌 Sky — Astronomical Events
Track upcoming solar eclipses, meteor showers, moon phases, planetary oppositions, and rare cosmic events — all with live countdowns. Includes:
- Live moon phase with illumination data (via USNO API)
- Planet visibility tracker (via Solar System OpenData API)
- Filterable event calendar by type (eclipses, meteors, comets, ISS, and more)

## 📡 External APIs Used

| API | Purpose | Key required |
|-----|---------|-------------|
| [USNO](https://aa.usno.navy.mil/data/api) | Live moon phase data | No |
| [wttr.in](https://wttr.in) | Moon phase fallback | No |
| [le-systeme-solaire.net](https://api.le-systeme-solaire.net) | Live planetary orbital data | No |
| [Google Fonts](https://fonts.google.com) | Typography | No |

All APIs are free, public, and require no authentication.
